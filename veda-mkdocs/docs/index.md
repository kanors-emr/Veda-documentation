# Veda 2.0 Documentation

**The complete data-management solution for energy-system models**

Veda 2.0 is a powerful data handling system for [The Integrated MARKAL-EFOM System (TIMES)](https://iea-etsap.org/index.php/etsap-tools/model-generators/times){ target="_blank" rel="noopener noreferrer" } — a bottom-up optimization model for energy-environment systems. Support for additional models like **OSeMOSYS** and **TEMOA** is in development.

Built with C#.NET and PostgreSQL, Veda 2.0 runs natively on Windows.
